# ⚠️ This server app has been superseded

The standalone Hono/Bun server in this directory has been replaced by **Next.js API Route Handlers** in `apps/web/src/app/api/`.

All routes are now co-located with the Next.js app and deploy to Vercel automatically — no separate server process required.

**If you want to run the standalone server separately** (e.g. on Railway/Render/Fly.io), it still works — but for Vercel deployment you only need `apps/web`.
